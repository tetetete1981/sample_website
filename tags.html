Important HTML tags

This page is designed as a quick reference for some of ... via table

The tables below don't include all HTML tags ....

Block elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
td Table data Include a table data cell
header header create and header space
div div add an div space
ol ol add an ordored list
ul ul add an unordored list

Inline elements

TAG Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> Technical sophistication
strong strong make strong <strong> at least a billion people</strong> Technical sophistication
a anchor Make hyperlink <a href="http://learnenough.com">Learn enough </a> Learn enough
img image add an image <img src="/images/tete.jpg" alt="tete"/>
span span add an span <span>Tete </span> Tetson
li li add an list element <li>hubert </li>
